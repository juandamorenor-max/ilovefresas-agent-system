# n8n

Carpeta para notas y exportaciones de n8n.

n8n funciona por ahora como puente de pruebas entre Telegram y Flowise. No debe ser base de datos ni fuente de verdad del pedido.

El flujo objetivo ya tiene contrato en:

```text
docs/n8n-dashboard-flowise-bridge.md
```

Resumen:

```text
Telegram/WhatsApp -> n8n -> backend/dashboard -> Flowise -> backend/dashboard -> n8n -> cliente
```
