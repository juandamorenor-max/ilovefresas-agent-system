# Politica de prompts

Estos archivos son la fuente local de verdad para los prompts de Flowise.

Cuando se haga una modificacion de prompts:

- guardar el prompt completo en este directorio;
- entregar al usuario el prompt completo, no solo el fragmento cambiado;
- evitar parches sueltos que despues sean dificiles de pegar en Flowise;
- mantener Flowise como copia manual de estos archivos hasta que exista import/export confiable.

Prompts activos principales:

- `router-central.md`
- `agente-pedido.md`
- `agente-datos.md`
- `agente-menu.md`
- `agente-general.md`
